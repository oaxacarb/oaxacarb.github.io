# Oaxaca Ruby User Group Website

## General information

This website is powered by [Middleman](http://middlemanapp.com).


## How to build the website?

* Clone the repository
* Install all the dependencies
```
bundle install
```
* Run the server
```
middleman server
```
* The website is in http://localhost:4567
* To deploy the site:
```
middleman deploy
```
